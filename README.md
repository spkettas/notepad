# Notepad demo

Electron 记事本，演示如何使用electron构建跨平台GUI。本程序基于electron v8.1.1 开发，electron各api版本互不兼容。




## API

* BrowserWindow
* Menu
* MenuItem
* dialog
* fs
* Clipboard
* ipcMain



## Building

```shell
$ git clone https://github.com/spkettas/notepad
$ cd notepad
$ npm install
$ npm start
```
